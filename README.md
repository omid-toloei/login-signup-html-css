## Login & Sign Up Page
A responsive **Login & Sign Up UI** built with **pure HTML and CSS**.  
This project is **UI-only** and does not include any backend or JavaScript functionality.

---

## Features
- Login & Sign Up forms  
- Fully responsive design  
- Clean and modern UI  
- Pure HTML & CSS  
- No JavaScript, no backend

---

## Built With
- HTML5  
- CSS3  

---

## Notes
This project focuses only on front-end layout and styling.  
Forms are **not functional** and are for UI demonstration purposes only.
